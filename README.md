### HR Analytics Dashboard

## Overview

The HR Analytics Dashboard is a Power BI project designed to help organizations analyze and visualize employee data to identify trends, improve retention, and make data-driven HR decisions.

## Objectives

Understand employee attrition patterns

Analyze workforce demographics and diversity

Identify performance and satisfaction trends

Support HR teams with actionable insights

## Key Insights & Features

Attrition Analysis: Track employee turnover by department, age group, and job role

Diversity Metrics: Gender distribution and average age across departments

Performance Overview: Department-wise ratings and job satisfaction scores

Interactive Filters: Easily filter by department, education, or years at company

KPIs: Key HR metrics such as retention rate, active employees, and average salary

## Tools & Technologies

Power BI Desktop – Data visualization & dashboard creation

Power Query – Data cleaning & transformation

DAX – Custom measures and calculated fields

Excel / CSV Dataset – Source data

## Workflow

1. Import dataset into Power BI

2. Clean and preprocess using Power Query

3. Create data model relationships

4. Build visuals, charts, and KPIs

5. Publish and share dashboard
